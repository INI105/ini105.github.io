本地模拟环境搭建

VSCode + LVGL8.3

相关软件获取

— —
Mingw64 x86_64-15.2.0-release-win32-seh-msvcrt-rt_v13-rev0.7z
CMake cmake-4.1.0-windows-x86_64.zip
SDL2 SDL2-devel-2.32.8-mingw.zip</ br>SDL3-devel-3.2.20-mingw.zip
lv_port_pc_eclipse lv_port_pc_eclipse-release-v8.3.zip
lv_drivers lv_drivers-be337d02ee797702f191499152db5f8c47e32d6e.zip
lvgl lvgl-0732400e7b564dd0e7dc4a924619d8e19c5b23a0.zip

x86_64-15.2.0-release-win32-seh-msvcrt-rt_v13-rev0.7z解压至C:\Program Files\mingw64\

cmake-4.1.0-windows-x86_64.zip解压至C:\Program Files\cmake\

目录C:\Program Files\mingw64\bin, C:\Program Files\cmake\bin\添加到Path环境变量

SDL2-devel-2.32.8-mingw.zip解压复制SDL2-2.32.8\x86_64-w64-mingw32文件夹到C:\Program Files\mingw64\覆盖

创建项目目录lvgl_v8.3

1)lv_port_pc_eclipse-release-v8.3.zip解压至lvgl_v8.3\

2)lv_drivers-be337d02ee797702f191499152db5f8c47e32d6e.zip解压至lvgl_v8.3\lv_drivers\

3)lvgl-0732400e7b564dd0e7dc4a924619d8e19c5b23a0.zip解压至lvgl_v8.3\lvgl\

4)SDL2-2.32.8\x86_64-w64-mingw32\bin\SDL2.dll 复制到 lvgl_8.3\

VSCode配置

插件:

C/C++
C/C++ Extension Pack

CMake
CMake Tools

CMakeLists 配置

->> build编译后生成lvgl_v8.3\bin\main.exe文件

注: 运行需要SDL2.dll文件

CodeBlocks + LVGL9.2

相关软件获取

— —
Code::Blocks codeblocks-25.03mingw-setup.exe
lv_port_win_codeblocks lv_port_win_codeblocks-release-v9.2.zip
lvgl lvgl-c903c1dc0fdfae20bce69d0a48894bdebae278ff.zip

创建项目目录lvgl_v9.2

lv_port_win_codeblocks-release-v9.2.zip解压至lvgl_v9.2\

lvgl-c903c1dc0fdfae20bce69d0a48894bdebae278ff.zip解压至lvgl_v9.2\lvgl\

->> `build`编译后生成`lvgl_v9.2\bin\Debug\LVGL.exe`文件
