HTML

Created2024-03-30|Updated2025-06-11|languageshtml
|Post Views:

HTML 指的是 超文本标记语言(Hyper Text Markup Language) , 它是用来描述网页的一种语言;

HTML不是一种编程语言, 而是一种 标记语言(Markup Language)；

标记语言是一套 标记标签(Markup Tag).

HTML语法规范

基本语法概述

  1. HTML标签是由< >包围的关键词, 例如<html>;

  2. HTML标签通常成对出现, 例如<html>和</html>, 称为双标签; 标签对中的第一个标签是开始标签, 第二个标签是结束标签;

  3. 有些特殊的标签是单个标签, 例如<br />, 称为单标签;

双标签关系可以分为两类: 包含关系和并列关系.

HTML基本结构标签

第一个HTML网页

每个网页都会有一个基本的结构标签, 页面内容也是在这些基本标签上书写;

HTML页面也称为HTML文档;

PS:

标签名 定义
<html></html> HTML标签
<head></head> 文档的头部
<title></title> 文档的标题
<body></body> 文档的主体

HTML文档的后缀名必须是.html或.htm, 浏览器的作用是读取HTML文档, 并以网页的形式显示.

文档类型声明标签

文档类型声明, 作用是告诉浏览器使用哪种HTML版本来显示网页.

PS:

含义: 当前页面采取的是HTML5版本来显示网页;

  1. <!DOCTYPE>声明位于文档中最前面的位置, 处于<html>标签之前;
  2. <!DOCTYPE>不是一个HTML标签, 是文档类型声明标签.

lang语言种类

用于定义当前文档显示的语言;

  1. en定义语言为英语;
  2. zh-CN定义语言为中文;

字符集

字符集(Character Set)是多个字符的集合, 以便计算机能够识别和存储各种文字.
在<head>标签内可以通过<meta>标签的charset属性来规定HTML文档应该使用哪种字符编码.

charset常用的值有: GB2312、BIG5、GBK和UTF-8, 其中UTF-8也被称为万国码, 基本包含了全世界所有国家需要用到的字符.

HTML常用标签

标签语义

即标签的含义, 起什么作用.

根据标签的语义, 在合适的地方给一个合理的标签, 可以让页面结构更为清晰.

标题标签 <h1>~<h6>(★)

HTML提供了6个等级的网页标题, 即<h1>~<h6>.

PS:

  1. 加了标题的文字会加粗, 字号依次减小;
  2. 一个标题独占一行.

段落和换行标签(★)

在HTML标签中, <p>标签用于定义段落, 可以将整个页面分为若干个段落.

PS:

  1. 文本在一个段落中会根据浏览器窗口的大小自动换行;
  2. 段落和段落之间保有空隙.

在HTML中, 一个段落中的文字会从左到右依次排列, 直到浏览器窗口的右端, 然后才自动换行; 如果希望某段文本强制换行显示, 可以使用换行标签<br />.

PS:

  1. <br />是一个单标签;
  2. <br />标签只是简单地开始新的一行, 跟段落不同, 段落之间会插入一些垂直的间距.

文本格式化标签

文本格式化标签, 使文字以特殊的方式显示.

语义 标签 优先使用 显示效果
*加粗 <strong></strong> or <b></b> <strong></strong> 加粗
*倾斜 <em></em> or <i></i> <em></em> 倾斜
删除线 <del></del> or <s></s> <del></del> 删除线
下划线 <ins></ins> or <u></u> <ins></ins> 下划线

<div> 和 <span> 标签

是一个”盒子”, 没有语义, 用来装内容.

PS:

  1. <div> 标签用来布局, 独占一行.
  2. <span> 标签用来布局, 一行上可以多个.

图像标签和路径

图像标签

<img> 标签用于定义 HTML 页面中的图像.

src 是 <img> 标签的必须属性[即特性], 用于指定图像文件的路径和文件名.

img 标签属性:

属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本[当图片不能显示时显示的内容]
title 文本 提示文本[当鼠标放上时显示的文字]
width 像素 图像的宽度
height 像素 图像的高度
border 像素 图像边框的粗细

PS:

  1. 图像标签可以拥有多个属性, 必须写在标签名后面;
  2. 属性之间不分先后顺序, 标签名与属性、属性与属性之间以空格分开;
  3. 属性采取键值对的格式, 即 属性 = “属性值”[key = “value”].

路径

目录文件夹: 普通文件夹, 存放页面所需的相关文件, 素材;

根目录: 打开目录文件夹的第一层即为根目录.

路径可以分为两种:

  1. 相对路径
  2. 绝对路径

超链接标签

HTML标签中, <a> 标签用于定义超链接, 作用是从一个页面链接到另一个页面.

链接的语法格式(anchor 锚)

href, target 属性的作用:

属性 作用
href 用于指定链接目标的 url 地址[ 必须属性 ]
target 用于指定链接页面的打开方式[ _self 默认值, 在当前窗口打开; _blank 在新窗口打开 ]

链接的分类

  1. 外部链接: e.g. <a href=”百度\">www.baidu.com">百度\
  2. 内部链接: 网站内部页面之间的相互链接, 直接链接内部页面名称. e.g. <a href=”index.html”>首页</a>
  3. 空链接: 没有确定链接目标. e.g. <a href=”#”>首页</a>
  4. 下载链接: 如果 href 地址是一个文件时, 会下载这个文件.
  5. 网页元素链接: 如文本、图像、表格、音频、视频等.
  6. 锚点链接: 点击链接, 可以快速定位到页面的某个位置.

锚点链接使用方式:

① href 属性中设置属性值为 #name 的形式; e.g. <a href=”#one”>示例</a>

② 目标位置标签, 添加属性: id=”name”. e.g. <h3 id=”one”>示例跳转</h3>

注释和特殊字符

注释

注释标签以 <!-- 开头, 以 --> 结束.

※注释用于理解和阅读代码, 程序不会执行注释内容.

特殊字符

特殊字符 字符代码 描述
&nbsp; &nbsp; 空格
< &lt;
> &gt;
& &amp;
¥ &yen;
© &copy;
® &reg;
° &deg;
± &plusmn;
× &times;
÷ &divide;
² &sup2;
³ &sup3;

表格标签

表格的主要作用

主要用于显示、展示数据.

表格的基本语法

① <table> </table> 标签用于定义表格;

② <tr> </tr> 标签用于定义表格中的行;

③ <th> </th> 标签用于定义表格中表头的单元格, 一般表头单元格位于表格的第一行或第一列, 其文本内容加粗居中显示;

③ <td> </td> 标签用于定义表格中的单元格;

表格属性

属性名 属性值 描述
align left、center、right 表格相对周围元素的对其方式
border 1 or “” 表格单元是否有边框, 默认为 "", 无边框
cellpadding px 单元格边框与内容之间的空白, 默认 1px
cellspacing px 单元格之间的空白, 默认 2px
width px or % 表格的宽度

※实际应用中, 利用css 来设置布局

表格结构标签

<thead>标签: 表格的头部区域, 内部必须有 <tr> 标签;

<tbody>标签: 表格的主体区域;

合并单元格

1.合并单元格方式

跨行合并: rowspan = “合并单元格的个数”

跨列合并: colspan = “合并单元格的个数”

2.目标单元格

跨行: 最上侧单元格为目标单元格

跨列: 最左侧单元格为目标单元格

3.合并单元格的步骤

①确定跨行/跨列合并;

②找到目标单元格, 合并方式 = “合并的单元格数量”; e.g. <td colspan = “2”></td>.

③删除多余单元格.

列表标签

列表主要用于布局, 根据使用场景, 列表可以分为三大类: 无序列表、有序列表和自定义列表.

无序列表<ul>

基本语法格式:

<ul>
<li>列表项1</li>
<li>列表项2</li>
…
</ul>

1.无序列表的各个列表项之间没有顺序级别, 是并列的;

2.<ul></ul> 中只能嵌套 <li></li>;

3.<li> 与 </li> 之间可以容纳所有元素;

4.无序列表自带有样式属性, 实际使用时利用 css 设置.

有序列表<ol>

基本语法格式:

<ol>
<li>列表项1</li>
<li>列表项2</li>
…
</ol>

1.有序列表的各个列表项排序以数字显示;

2.<ol><ol> 中只能嵌套 <li></li>:

3.<li> 与 </li> 之间可以容纳所有元素;

4.有序列表自带有样式属性, 实际使用时利用 css 设置.

自定义列表<dl>

自定义列表常用于对术语或名词进行解释和描述, 定义列表的列表项前没有任何项目符号.

基本语法格式:

<dl>
<dt>名词</dt>
<dd>解释1</dd>
<dd>解释2</dd>
…
</dl>

1.<dl></dl> 中只能包含 <dt> 和 <dd>;

2.<dt> 和 <dd> 个数没有限制, 多数情况下一个 <dt> 对应多个 <dd>;

3.<dt> 与 <dd> 可以容纳所有元素;

4.实际使用时利用 css 设置属性.

表单标签

目的: 进行交互, 收集信息.

表单的组成

一个完整的表单通常由表单域、表单控件(又表单元素)和提示信息三部分构成.

表单域<form>

表单域是一个包含表单元素的区域.

基本语法格式:

<form action=”url地址” method=”提交方式” name=”表单域名称”>
表单元素控件
</form>

属性:

属性 属性值 作用
action url 地址 用于指定接收并处理表单数据的服务器程序的 url 地址
method get / post 用于设置表单数据的提交方式
name 名称 用于指定表单的名称, 以区分同一个页面中的多个表单域

表单控件(表单元素)

<input> 表单元素

<input> 标签用于收集用户信息, 其包含一个 type 属性, 输入字段拥有多种形式(文本、复选框、掩码后的文本、按钮、单选按钮等).

语法格式:

<input type=”属性值” />

①<input /> 标签为单标签;

②type 属性用来指定不同的控件类型.

type 属性:

属性值 描述
button 定义可点击按钮
checkbox 定义复选框
file 定义输入字段和”浏览”按钮, 供文件上传
hidden 定义隐藏的输入字段
image 定义图像形式的提交按钮
password 定义密码字段(该字段中的字符被掩码)
radio 定义单选按钮
reset 定义重置按钮(会清除表单中的所有数据)
submit 定义提交按钮(把表单数据发送到服务器)
text 定义单行的输入字段(可输入文本, 默认宽度20个字符)

<lable> 标签

为 input 元素定义标注. 用于绑定一个表单元素, 当点击 <lable> 标签内的文本时, 浏览器自动将焦点(光标)转到或选择对应的表单元素上.

语法格式:

<label for=”sex”>男</ label>

<input type=”radio” name=”sex” id=”sex” />

※ <label> 标签的 for 属性与相关元素的 id 属性相同.

<select> 表单元素

定义下拉列表.

语法格式:

<select>
<option>选项1</option>
<option>选项2</option>
<option>选项3</option>
…
</select>

1.<select> 中至少包含一对 <option>.

2.在 <option> 中定义 selected=”selected” 时, 此项即为默认选中项.

<textarea> 表单元素

用于定义 多行文本输入, 常见于留言板、评论.

语法格式:

<textarea rows=”3” cols=”20”>
文本内容
</textarea>

1.通过 <textarea> 标签可以创建多行文本输入框.

2.属性 cols=”每行中的字符数”, rows=”显示的行数”, 实际开发使用 CSS 定义.
